# Data Folder

This folder is reserved for dataset references and sample inputs used in this project.

## Data Availability
The full datasets are not included in this public repository due to:
- file size constraints
- licensing and source restrictions
- repository cleanliness and maintainability

## Datasets Used
The project combines multiple news datasets:
- BBC News (Kaggle)
- Reuters-21578 (Kaggle)
- AG News Corpus (Kaggle)
- Scraped articles from:
  - CNN
  - The Guardian
  - TechCrunch
